# Sliding Menu App

## How to Set Up the App

1. First, navigate to the `server` directory:

   ```bash
   cd server

2. Install dependencies for the server:
  npm install

3. Start the server:
  node index.js

4. Go back to the root directory of the app and install its dependencies:
  npm install

5. Launch the Expo app:
  expo start
